# Example datasets

## Wine Data Set

`wine1991.csv`

These data are the results of a chemical analysis of wines grown in the same region in Italy but derived from three different cultivars. The analysis determined the quantities of 13 constituents found in each of the three types of wines. [Download](https://archive.ics.uci.edu/ml/datasets/Wine) data from UCI archive.

Columns contain measurements of 13 wine features. Column names describe feature names.

Rows contain individual wines. The first column contains wine ID in the format [cultivar class]_[wine ID].

Data Set Characteristics: Multivariate
Number of samples: 178
Number of features/attributes/measurements: 13
Attribute characteristics: Integer, Real
Missing Values? No

Source:

Forina, M. et al., *PARVUS - An Extendible Package for Data Exploration*, Classification and Correlation. Institute of Pharmaceutical and Food Analysis and Technologies, Via Brigata Salerno, 16147 Genoa, Italy. 

## Wine Quality Data Set

`winequality-red.csv`

These data are the results of a physicochemical analysis of red variants of the Portuguese "Vinho Verde". The analysis determined the quantities of 11 features found in each of the wines. [Download](https://archive.ics.uci.edu/ml/datasets/Wine+Quality) data from UCI archive.

Columns contain measurements of 11 wine features. Column names describe feature names.

Rows contain individual wines. The first column contains wine ID in the format [quality score]_[wine ID]. The quality score is a sensory variable (the output) between 0 and 10.

Data Set Characteristics: Multivariate
Number of samples: 4898
Number of features/attributes/measurements: 11
Attribute characteristics: Real
Missing Values? No
